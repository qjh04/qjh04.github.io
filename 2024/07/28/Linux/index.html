第一章 初识Linux

第一节 操作系统概述

用户和计算机硬件之间的桥梁，调度和管理硬件进行工作。

第二节 初识Linux

组成：

  • Linux系统内核
    • 提供最核心的功能，如cpu调度、内存调度等
  • 系统级应用程序
    • 可理解为出厂自带软件，供用户快速上手，如文件管理器、视频播放器等

第三节 虚拟机

借助虚拟化技术，通过软件模拟计算机硬件，并给虚拟硬件安装操作系统。这样，就可以在电脑上安装一个虚拟操作系统。

小知识：ncpa.cpl 命令可以打开网络适配器查看页面。

第四节 虚拟机快照

快照可以将当前虚拟机的状态保存下来，在以后可以通过快照恢复到前面所保存的状态。

第二章 Linux基础命令

第一节 Linux的目录结构

树形结构，没有盘符概念，只有一个根目录 / ，所有文件都在它之下。

第二节 Linux命令入门

命令：即Linux程序，一个命令就是一个程序。

命令行：即Linux终端，是一种命令提示符界面，以纯字符的形式操作系统，可以使用各种字符化命令对系统发出指令。

通用格式： command [-potions] [parameter]

第三节 ls命令入门

语法： ls [-a -l -h] [Linux路径]

不加选项，会展示出当前目录下（默认home目录）的内容，以平铺格式显示。

每一个用户在Linux系统的专属目录,默认在:/home/用户名

Linux命令行在执行命令的时候,需要一个工作目录,打开命令行程序(终端)默认设置工作目录在用户的HOME目录

选项解释：

  • -a ：表示all，所有文件，可以看到隐藏文件夹
  • -l ：从横向平铺变成竖向的列表
  • -h ：方便阅读，会列出单位K、M、G

第四节 目录切换命令cd、pwd

语法：cd [Linux路径]

不写参数，切换到home目录。

语法：pwd

打印当前工作目录路径

第五节 相对路径、绝对路径和特殊路径

  • 绝对路径：以根目录为起点
  • 相对路径：以当前所在目录作为起点
  • 特殊路径符：. 表示当前目录 .. 表示上一级目录 ~表示home目录

第六节 创建目录命令 mkdir

语法：mkdir [-p] Linux路径

选项-p表示可以自动创建不存在的父目录，即可以创建多层级目录

第七节 文件操作命令

1、touch创建文件

语法：touch Linux路径

2、cat查看文件

语法： cat Linux路径

3、more查看文件

语法： more Linux路径

与cat不同，多行可以翻页，通过空格翻页，通过q退出

4、cp复制文件文件夹

语法： cp [-r] 参数1 参数

  • 选项-r：当复制文件夹时使用，表示递归
  • 参数1：被复制的文件或文件夹
  • 参数2：要复制去的地方

5、mv移动文件文件夹

语法： mv 参数1 参数2

  • 参数1：要移动的文件或文件夹
  • 参数2：要移动到的地方

6、rm删除文件文件夹

语法： rm [-r -f] 参数1 参数2 参数、、、

  • 同cp命令一样，-r表示删除文件夹
  • -f表示强制删除
  • 参数表示要删除的文件或文件夹路径
  • 支持通配符*，表示任何内容

第八节 查找命令

1、which

语法： which 要查找的命令名

前面所学的命令，本质上都是一个二进制程序，可以通过switch命令，查看使用的一系列命令程序文件

2、find

语法： find 起始路径 -name 被查找文件名

通过find命令去搜索指定的文件，也可以使用通配符*，

按文件大小查找文件

语法： find 起始路径 -size +|-n[kmg]

  • +、-表示大于和小于

  • n表示大小数字

  • kMG表示大小单位,k(小写字母)表示kb,M表示MB,G表示GB

第九节 grep、wc和管道符

1、grep

可以通过grep命令,从文件中通过关键字过滤文件行。

语法: grep [-n] 关键字 文件路径

  • 选项-n,可选,表示在结果中显示匹配的行的行号。
  • 参数,关键字,必填,表示过滤的关键字,带有空格或其它特殊符号,建议使用”将关键字包围起来
  • 参数,文件路径,必填,表示要过滤内容的文件路径,可作为内容输入端口

2、wc

可以通过wc命令统计文件的行数、单词数量等

语法： WC [-c -m -1 -w] 文件路径

  • 选项,-C,统计bytes数量

  • 选项,-m,统计字符数量

  • 选项,-l,统计行数

  • 选项,-w,统计单词数量

  • 参数,文件路径,被统计的文件,可作为内容输入端口

3、管道符 |

将左边的结果 作为 右边的输入

第十节 echo、tail和重定符

1、echo

可以用echo命令在命令行界面输出想输出的内容

语法： echo 字符串

如果字符串里有特殊符号，需要双引号引起来

反引号 ‘ ’ ：可以被看做转义字符，引号里的字符可以被看做是命令执行

2、重定向符

  • “>” ：将左侧的结果覆盖写入到右边的文件中
  • “>>”：将左侧的结果追加写入到右边的文件中

3、tail命令

使用tail命令,可以查看文件尾部内容,跟踪文件的最新更改,语法如下:

tail [-f -num] Linux路径

  • 参数,Linux路径,表示被跟踪的文件路径

  • 选项,-f,表示持续跟踪

  • 选项,-num,表示,查看尾部多少行,不填默认10行

第十一节 vi、vim编辑器

  • 命令模式：所有按键被视为命令，以命令驱动执行不同的功能，不能自由进行文本编辑
  • 输入模式：所谓的编辑模式，可以对文本进行编辑
  • 底线命令模式：以：开始，通常用于保存或退出

第三章 Linux用户和权限

第一节认识root用户

超级管理员，拥有操作系统最大操作权限

切换用户

语法： su [-] [用户名]

用于账户切换的系统命令，-：表示切换的同时加载环境变量，不填用户名默认切换到root，exit退出

sudo命令

sudo 其他命令

为普通命令授权，临时以root身份执行，需要为普通用户配置sudo认证才可以使用

第二节用户和用户组

Linux可以配置多个用户，多个用户组，一个用户可以进多个用户组。可以针对用户进行权限管控，也可以对用户组进行管控

用户组管理

  • 创建用户组：groupadd 用户组名
  • 删除用户组：groupdel 用户组名

用户管理

  • 创建用户：useradd [-g -d] 用户名
    • 选项 :- g指定用户的组,不指定-g,会创建同名组并自动加入,指定-g需要组已经存在,如已存在名组,必须使用-g
    • 选项 :- d指定用户HOME路径,不指定,HOME目录默认在:/home/用户名
  • 删除用户：userdel [-r] 用户名
    • 选项：-r表示删除用户的home目录
  • 查看用户所属组：id[用户名]
    • 选项用户名不加 则查看当前用户的所属组
  • 修改用户所属组：usermod -aG 用户组 用户名
    • 将指定用户加入用户组
  • 查看所有用户或用户组：getnet passwd / group

第三节 查看权限控制

r：可读 w：可写 x：可执行

修改权限信息

语法：chmod [-R] 权限 文件或文件夹

选项-R表示对文件夹内部的所有文件应用同样的权限，权限可以用二进制表示rwx

修改用户用户组权限

语法：chown [-R] [用户] [:] [用户组] 文件文件夹

此命令只适用于root用户，将文件文件夹修改为某用户用户组，-R同上

第二节 安装软件

yum命令

RPM包软件管理器,用于自动化安装配置Linux软件,并可以自动解决依赖问题。

语法:yum [-y] [install | remove | search] 软件名称

  • 选项 :- y,自动确认,无需手动确认安装或卸载过程

  • install:安装

  • remove:卸载

  • search:搜索

yum命令需要root权限哦,可以su切换到root,或使用sudo提权，yum命令需要联网

systemctl命令

Linux系统很多软件(内置或第三方)均支持使用systemctl命令控制:启动、停止、开机自启

能够被systemctl管理的软件,一般也称之为:服务

语法: systemctl start | stop| status | enable| disable 服务名

  • start 启动
  • stop关闭
  • status查看状态
  • enable开启开机自启
  • disable 关闭开机自启

第三节 软连接

类似windows中的快捷方式

语法：ls -s 参数1 参数2

  • -s选项,创建软连接

  • 参数1:被链接的文件或文件夹

  • 参数2:要链接去的目的地

第四节 日期和时区

通过date命令可以在命令行中查看系统的时间

语法:date [-d] [+格式化字符串]

  • -d按照给定的字符串显示日期,一般用于日期计算，+号后边可以加day、month、year
  • 格式化字符串:通过特定的字符串标记,来控制显示的日期格式
    %Y 年
    %y 年份后两位数字(00.99)
    %M 月份(01.12)
    %d 日 (01.31)
    %H 小时(00.23)
    %M 分钟(00.59)
    %s 秒(00.60)
    %s 自 1970-01-01 00:0000 UTC 到现在的秒数

修改时区

  • rm -f /etc/localtime
  • sudo ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

也可以用ntp程序自动校准

第五节 ip地址和主机名

命令：ifconfig 和 hostname

第六节 网络传输

ping

可以通过ping命令,检查指定的网络服务器是否是可联通状态

语法:ping [-c num] ip或主机名

  • 选项 :- c,检查的次数,不使用-c选项,将无限次数持续检查

  • 参数:ip或主机名,被检查的服务器的ip地址或主机名地址

wget

wget是非交互式的文件下载器,可以在命令行内下载网络文件

语法:wget [-b] ur1

  • 选项 :- b,可选,后台下载,会将日志写入到当前工作目录的wget-log文件

  • 参数:url,下载链接

curl

curl可以发送http网络请求,可用于:下载文件、获取信息等

语法:curl [-0] ur1

  • 选项 :- 0,用于下载文件,当url是下载链接时,可以使用此选项保存文件

  • 参数:url,要发起请求的网络地址

nmap

可以通过Linux命令去查看端口的占用情况，使用nmap命令,安装nmap:yum-y install nmap

语法:nmap 被查看的IP地址

netstat

查看端口占用情况

语法:netstat -anp | grep 端口号,安装netstat:yum-y install net-tools

第七节 进程管理

ps

查看进程信息

语法： ps [-e -f]，一般来说两个选项都需要用，会展示出详细信息

kill

关闭进程

语法： kill [-9] 进程ID，选项-9表示强制关闭

第八节 主机状态

top

查看cpu、内存占用状态

df [-h]

查看磁盘使用情况，-h会显示单位

iostat

可以使用iostat查看CPU、磁盘的相关信息

语法:iostat [-x] [num1] [num2]

  • 选项 :- x,显示更多信息

  • num1:数字,刷新间隔,num2:数字,刷新几次

sar

可以使用sar命令查看网络的相关统计(sar命令非常复杂,这里仅简单用于统计网络)

语法:sar -n DEV num1 num2

  • 选项 :- n,查看网络,DEV表示查看网络接口

  • num1:刷新间隔(不填就查看一次结束),num2:查看次数(不填无限次数)

第九节 环境变量

记录一些关键性信息，辅助系统运行，是一种键值对型结构，env命令可查看环境变量

取到环境变量语法： echo $PWD

设置环境变量语法： export key=value，如果设置永久生效需要修改配置文件

第十节 压缩和解压

两种文件格式：.tar 简单封装，体积不会有太大变化，.gz 极大减少压缩后的体积

tar命令常用组合

压缩：

  • tar -cvf test.tar t1.txt t2.txt 选项解释：压缩 显示 创建的文件
  • tar -zcvf test.tar.gz t1.txt t2.txt 选项解释：gzip格式 压缩 显示 创建的文件

解压：

  • tar -xvf test.tar 选项解释：解压 显示 要解压的文件
  • tar -xvf test.tar -C Linux文件路径 选项解释：解压 显示 要解压的文件 加一个目标路径
  • tar -zxvf test.tar.gz 选项解释：gzip格式 解压 显示 要解压的文件

zip压缩格式

压缩：

  • 命令： zip [-r] 参数1 参数2 参数3 ……
  • 有文件夹时，加上 -r

解压：

  • 命令： unzip [-d] 参数
  • -d 表示要解压到的地方，后边接路径
